## 下载

### Docker

```
docker pull apachecn0/flygon-prog-wiki
docker run -tid -p <port>:80 apachecn0/flygon-prog-wiki
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-prog-wiki
flygon-prog-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
